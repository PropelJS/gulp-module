These metrics are not meant to be used as law but are a relative measure of the quality and cost of a program

Metric Value Description
Average Maintainability 81.08 This is the relative maintainability of a program on a scale of 0-100. Higher is better.
Average Difficulty 8.62 This is a measure of how difficult it is to understand the code. Lower is better.
Total Cyclomatic Complexity 43 This is how complex the code is based on the number of paths through the code. Lower is better.
Average Cyclomatic Complexity 1.23 This is the total complexity for the program from above.
Average Cyclomatic Density 5.59 This is a re-expression of the above metric as a percentage based on the lines of code. Lower is better.
Average Effort 10908.27 This is the amount of effort needed to write the code based on it's difficulty and length
Total Effort 381789.61 This is the total of the effort needed across the program.
Average Time 0.17 Hours This is the estimated amount of time needed to write the software based on the effort required.
Total Time 5.89 Hours This is the estimated amount of time needed to write the software based on the effort required.
Average Cost $7.88 This is the estimated cost to develop the software based on the time needed to develop it.
Total Cost $275.74 This is the estimated cost to develop the software based on the time needed to develop it.
Average Bugs 0.30 This is the estimated number of bugs per file based on their length and complexity.
Total Bugs 10.42 This is the estimated number of bugs in the software.
Average Logical SLOC 27 This is the number of lines of software that are considered logic.
Total Logical SLOC 934 This is the number of lines of software that are considered logic.
Average Physical SLOC 57 This is the total lines of source code including up to 25% whitespace, but ignoring excessive formatting.
Total Physical SLOC 1998 This is the total lines of source code including up to 25% whitespace, but ignoring excessive formatting.
Average Dependencies 1.60 This is the average number of dependencies per file in the program.
Total Dependencies 56 This is the total Number of dependencies in the software.)
